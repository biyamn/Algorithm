# 내 풀이

입력받는 부분만 알겠다.. DFS/BFS를 사용한다는 것도 알겠는데, 어떻게 구현해야 할지 모르겠다.

```python
n, m = map(int, input().split(' '))

map = []
for i in range(n):
  line = input()
  map.append(line)
print(map)
```
